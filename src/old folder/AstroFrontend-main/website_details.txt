## 3. USER DASHBOARD & PRIVATE PAGES

### A. Dashboard Sidebar (Aside)
- **Profile Summary**: Quick view of user name and profile picture.
- **Navigation Links**: Fast access to My Profile, Enrolled Courses, Add Courses, and Payments history.
- **Responsive Drawer**: Mobile-optimized side menu for dashboard navigation.

### B. Enrolled Courses Page
1. **Courses Overview**: List of all purchased free and paid courses.
2. **Validity Tracker**: Visual Pie Charts showing "Days Used" vs "Days Left" for paid subscriptions.
3. **Learning Access**: "Start Learning" triggers that open specific course content.
4. **Resource Management**: Access to video sessions and downloadable study materials.

### C. Student Profile Page
1. **Profile Header**: Bold "Student Profile" banner.
2. **Detail View**: Displays Full Name, Father's/Mother's Name, DOB, and Email.
3. **Edit Mode**: Allows users to upload a new profile picture to Firebase Storage and update personal information in Firestore.

## 4. ADMIN PANEL (MANAGEMENT)
- **Article Manager**: Add, edit, and arrange blog posts.
- **Course Manager**: Upload new courses, modules, and video links.
- **Inquiry Tracker**: View and respond to contact form submissions and appointments.
- **Payment Monitoring**: Full list of transactions, subscriptions, and EMI logs.
- **Live Session Control**: Manage Jitsi meeting links and schedules.

## 5. WEBSITE FLOW & LOGIC
1. **Discovery**: Public exploration via Home -> Services -> Articles.
2. **Account Creation**: Users sign up (Firebase) and create a profile.
3. **Purchase/Enrollment**: Course selection -> Status check (Firestore) -> Payment gateway -> Enrollment.
4. **Consuming Content**: Dashboard -> Enrolled Courses -> Start Learning -> Video Player/Materials.
5. **Support Flow**: Use of Floating WhatsApp or Appointment form for direct expert contact.
